# Alchiromancy

> **Game design document for a potential Minecraft mod.**
> This is *not* a mod (yet) — it's a collection of design notes describing a magic & alchemy system that may, one day, become one.

A playthrough-spanning system built on two opposing forces — **Magic** (create / transform) and **Alchemy** (transmute / amplify) — that must serve one another. The player who balances both is an **Alchiromancer**.

Start with the index — [`EN/Index.md`](EN/Index.md) — or jump straight to [`EN/Concept.md`](EN/Concept.md) and [`EN/Glossary.md`](EN/Glossary.md).

## Status

🚧 **Design phase.** No code, no playable build. Everything here is concept, lore and mechanics on paper.

## A note on the files

These notes are **written by the author in French**. The English files (in the `EN/` folder) are **AI-translated** for GitHub and may contain wording inaccuracies — the French versions are the source of truth.

Three early notes — `Concept`, `Magic` and `Magic and Alchemy` — predate the detailed design and may be **outdated**.
